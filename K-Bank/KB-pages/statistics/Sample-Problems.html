2.3.7

Question

A widget is being made at a factory. The process involves sanding them, analyzing the thickness, then either approving them or use, rejecting them, or sanding them a second time. The following probabilities apply to this process:

  • After the first sanding, 70% meet specification and are accepted, 10% are rejected and scrapped, and 20% are sent for a second sanding.
  • Of those that get sent for a second sanding, 90% are accepted and 10% are scrapped after this second sanding.
  1. What is the probability a widget is sanded only once?
  2. Given that a widget is not re-sanded, what is the probability that it is scrapped?
  3. What is the probability that a widget is scrapped?
  4. Given that a widget is scrapped, what is the probability that is was sanded twice?
  5. What is the probability that a widget is accepted at all? (So after either the first or second sanding.)
  6. For an accepted widget, what is the probability that it was sanded twice? (There are two opportunities for acceptance.)
  7. For an accepted widget, what is the probability that it was sanded only once?

Answer

Write out a tree diagram and a Ven diagram or block diagram to help visualize the situations.

We will make the following letter designations:

  • $S_1=$ first sanding
  • $r_1=$ rejected on first sanding
  • $a_1=$ accepted on first sanding
  • $S_2=$ sent for a second sanding
  • $r_2=$ rejected on second sanding
  • $a_2=$ accepted on second sanding

  1. $P(S_2^c)=?$

    $=P(m_1 \cup r_1) = P(m_1)+P(r_1)=0.8$

    This works because there is no overlap between $m_1$ and $r_1$ events.

  2. $P(r_1|S_2^c)=?$

    $=\frac{P(r_1 \cap S_2^c)}{P(S_2^c)}=\frac{P(r_1)}{P(S_2^c)}=\frac{0.1}{0.8} = 0.125$

    Be careful about what we are being asked. We aren't being asked what the chance of being rejected the first time is (10%). We are being asked that if we happen to have a widget that is definitely not going to be sanded again, what is the chance that it is getting rejected. Notice how in the second situation, we have injected additional information into the situation, additional knowledge that is not present in the simpler, 10%.

    The second thing to notice is that $P(r_1 \cap S_2^c)$ is just equal to $P(r_1)$. To see this, look at the block diagram and notice trace out the two regions. The overlap of $r_1$ with everything that isn't $S_2$ is just $r_1$. Remember that everything that isn't $S_2$ is just a round-a-bout way of referring to everything that is $S_1$.

  3. $P(r_1 \cup r_2)=$?

    $=P(r_1) + P(r_2|S_2)P(S_2) = 0.1 + (0.1)(0.2) = 0.12$

    When break up the classic union probability in a classic ven diagram, we have $P(A)+P(B)-P(A\cap B)$, but in this case, there is no overlap and so no intersection term. In addition, there is a "conversion" which must take place since we cannot work directly with $P(r_2)$ as that would, presumably, refer to the 10%. However, that 10% is in reference to being within $S_2$ so we use the term shown to account for that.

    Another way to get to view the change in the term is from the tree diagram. We want to talk about $r_2$ but to get to it, we must first pass through $S_2$. So that location really refers to two events taking place, $r_2$ and $S_2$ or $r_2\cap S_2$. So we write $P(r_1)+P(r_2\cap S_2)$ and then use the definition of the conditional to change the second term to what we used above.

    An alternative approach to this problem is that: $P(r_1\cup r_2) = 1- P(r_1^c\cap r_2^c)$. The intersection here includes everything except $r_1$ and $r_2$. Which only leaves us with the two acceptance regions. Therefore this term is equal to:

    $P(r_1^c\cap r_2^c) = P(a_1) + P(a_2|S_2)P(S_2)$ which we could then do a similar intersection notation instead of conditional if we wanted to. This comes out to 0.88, which when subtracted from 1 gives the correct answer of 0.12.

  4. We phrase this one as $P(S_2|r_1\cup r_2)=?$.

    $=\frac{P(S_2\cap(r_1\cup r_2))}{P(r_1\cup r_2)}$

    The intersection of $S_2$ with the union of $r_1$ and $r_2$ is just $r_2$. But to get to $r_2$, we see from our tree diagram or the ven diagram, that we have to go through $S_2$, so

    $P(S_2\cap(r_1\cup r_2))=P(r_2|S_2)P(S_2)$

    Repeating the previous line, this leaves us with

    $=\frac{P(S_2\cap(r_1\cup r_2))}{P(r_1\cup r_2)}= \frac{P(r_2|S_2)P(S_2)}{P(r_1\cup r_2)} = \frac{(0.1)(0.2)}{(0.12)} = 0.167$

  5. $P(a_1\cup a_2)=?$

    From previous reasoning on part c with the similar rejection-based question, we have

    $=P(a_1)+P(a_2|S_2)P(S_2) = 0.7 + (0.9)(0.2) = 0.88$

    Alternatively, we can recognize that for a widget to be accepted at all, it has to be never rejected. This means we can write

    $=1-P(r_1\cup r_2)$

    which we have already solved in part c. Note that this is not a case where we can use the compliment to negate the $a_1$ and $a_2$ events. The complement of $a_1$ also contains the event $S_2$, not only $r_1$. So we should be careful. Certainly reject is the opposite of accept in common usage, but here they are just names for our events so we must be careful.
